Здесь собраны сокращения единиц измерения, которые часто встречаются в интерфейсах.
Можно посмотреть, [на что мы опирались](#справочники) при выборе.

Постарайтесь использовать одинаковые сокращения в одном проекте, а лучше — во всех сервисах.
Обратите внимание на пунктуацию: многие сокращения пишутся без точки, и это правильно.

## Числа

Если при сокращении числа отсекается часть, точка ставится. Если выбрасываются гласные буквы, точка не нужна.

| Сокращение | Значение | Пример           |
| ---------- | -------- | ---------------- |
| тыс.       | тысяча   | 100 тыс. человек |
| млн        | миллион  | 1 млн яблок      |
| млрд       | миллиард | 2 млрд жителей   |
| трлн       | триллион | 7 трлн звезд     |

## Единицы измерения

Многие единицы измерения в сокращенном виде пишутся без точки (см. [ГОСТ 8.417—2002](https://ru.wikisource.org/wiki/ГОСТ_8.417—2002)).

| Сокращение    | Значение        | Пример                                            | Комментарий                                                             |
| ------------- | --------------- | ------------------------------------------------- | ----------------------------------------------------------------------- |
| м             | метр            | Глубина колодца 12 м, и это не предел.            | Так же с любой приставкой, например _см_ и _км_                         |
| м<sup>2</sup> | метр квадратный | Площадь комнаты — 255 м<sup>2</sup>, без балкона. | Так же с любой приставкой, например _см<sup>2</sup>_ и _км<sup>2</sup>_ |
| г             | грамм           | 800 000 г — столько весит жираф.                  | Так же с любой приставкой, например _мг_                                |
| кг            | килограмм       | 800 кг — столько весит жираф.                     |                                                                         |
| т             | тонна           | 2 т — средний вес слона.                          |                                                                         |
| л             | литр            | 2 л — всё, что тебе нужно.                        | Так же с любой приставкой, например _мл_                                |

| мин        | минута   | 12 минут — всё, что тебе осталось. |                                                                                                                                                              |
| с          | секунда  | 12 c                               | Так же с любой приставкой, например _мкс_                                                                                                                    |

## Дни недели

В дизайн-системе для дней недели используются **только двухбуквенные сокращения**.

| Сокращение | Значение    |
| ---------- | ----------- |
| Пн         | понедельник |
| Вт         | вторник     |
| Ср         | среда       |
| Чт         | четверг     |
| Пт         | пятница     |
| Сб         | суббота     |
| Вс         | воскресенье |

В таблицах, графиках и контролах лучше писать день с прописной буквы,
если он стоит в начале строки. Если слово не первое, то со строчной.

## Месяцы

## Валюта

| Сокращение | Значение | Пример  | Комментарий                                                                                                                                              |
| ---------- | -------- | ------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `₽`        | рубль    | `100 ₽` | Знак рубля пишется после суммы, перед ним пробел. Если знак рубля не получается использовать, пишите _р._ или _рублей_ (в этом случае слово склоняется). |
| `$`        | доллар   | `$100`  | Знак доллара перед суммой, без пробела. Со знаком евро пишем по такому же принципу.                                                                      |
