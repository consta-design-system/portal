Дизайн-система Consta — это набор библиотек готовых интерфейсных компонентов для создания цифровых продуктов и сервисов любого уровня сложности.

Наша дизайн-система состоит из нескольких библиотек компонентов с документацией на русском языке. Компоненты реализованы в двух форматах:

- [репозиторий](https://github.com/consta-design-system) с открытым исходным кодом на React,
- библиотеки в [Figma Community](https://www.figma.com/@Consta).

Consta распространяется бесплатно по [лицензии MIT](https://github.com/consta-design-system/uikit/blob/master/LICENSE).

# Принципы дизайн-системы

## Открытость

Consta — открытая дизайн-система. Вы можете использовать её как для личных, так для рабочих проектов любой организации.

## Универсальность

Consta можно использовать при создании проектов разного масштаба: от инструментов мониторинга и управления оборудованием до интранет-порталов и личных кабинетов в широком спектре отраслей.

## Гибкость

Consta предлагает готовые блоки, большой набор адаптивных компонентов и мобильные UI-киты. Мы стилизовали популярные графики, диаграммы и таблицы, чтобы работа с данными была удобнее.

Несмотря на то, что в дизайн-системе много компонентов, всегда найдётся что-то особенное, чего не хватит именно вашему проекту. Мы стараемся добавить в дизайн-систему самые распространённые и популярные компоненты и не претендуем на полноту: вы можете самостоятельно добавить в свой проект то, чего вам не хватает.

# Технологии

Дизайн-система построена по принципам [БЭМ-методологии](https://ru.bem.info/), на основе библиотеки [whitepaper](https://github.com/whitepapertools). Библиотеки компонентов дизайн-системы существуют в React-реализации.

Технологический стек:

- [React](https://reactjs.org/)
- [Typescript](https://www.typescriptlang.org/)
- PostCSS ([postcss-nested](https://github.com/postcss/postcss-nested), [postcss-color-function](https://github.com/postcss/postcss-color-function), [postcss-simple-vars](https://github.com/postcss/postcss-simple-vars) для media-query)
- [whitepaper](https://github.com/whitepapertools)

Библиотеки рассчитаны на браузеры семейства Chromium.

# Тематизация

В Consta есть три цветовые схемы: светлая, тёмная и акцентная. Вы можете создать и свою тему, прописав в переменных базовые цвета — остальные подбираются автоматически. В одном продукте можно использовать несколько тем как для всего интерфейса, так и для отдельных его блоков.

[Подробнее о темах](##LIBS.LIB.STAND/lib:portal/stand:theme-themeabout)

# Семантические переменные

Все параметры темы описаны с помощью переменных [CSS Custom Properties](https://developer.mozilla.org/ru/docs/Web/CSS/Using_CSS_custom_properties) с семантическими именами. Значения цветов, отступов и размеров типографики берутся из темы, а значит, легко меняются вместе с ней. Например, переменная --color-typo-link не должна использоваться для фона блока, а переменная --color-bg-warning используется только для отображения статуса.

# Кастомные блоки и компоненты

Благодаря гибкости в настройке компонентов Consta, можно быстро собирать кастомные блоки и элементы интерфейса, сохраняя при этом консистентность всех продуктов компании. Наша команда всегда на связи и готова помочь в реализации.
