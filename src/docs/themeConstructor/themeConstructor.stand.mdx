Конструктор тем помогает создавать собственные [темы](##LIBS.LIB.STAND/lib:portal/stand:theme-themeabout) без навыков программирования и отслеживать результат изменений в режиме реального времени. С его помощью вы сможете:

- [Настраивать токены](#настройка-токенов) (цвета, отступы, шрифты, тени и радиусы).
- [Сохранять](#сохранить-изменения) и [скачивать](#скачать-тему) темы в формате CSS или JSON.

# Как работать в конструкторе

## Создать тему

1. Перейдите в [конструктор тем](##THEME_CONSTRUCTOR).
2. В открывшемся окне выберите нужное действие:
   - **Создать тему**.
   - **Восстановить сессию**, если хотите продолжить настройку темы с прошлой сессии.
   - **Открыть тему**, если хотите продолжить настройку одной из сохраненных тем.

## Настроить тему

1. Выберите вкладку с нужным токеном.
2. Измените [настройки токенов](##LIBS.LIB.STAND/lib:portal/stand:theme-constructor/hash:настройка-токенов).

## Сохранить изменения

После сохранения изменений вы можете продолжить настраивать тему и после закрытия вкладки конструктора.

> Темы автоматически сохраняются в кэше браузера. Очистка кэша ведет к потере сохраненных тем.

1. На верхней панели нажмите кнопку **Сохранить**.
2. В открывшемся окне выберите, куда сохранить изменения:
   - **В новую тему**. Останьтесь на вкладке **В новую тему** и введите название темы.
   - **В существующую тему**. Перейдите на вкладку **В существующую тему** и в раскрывающемся списке выберите нужную для сохранения изменений тему.
3. Нажмите **Сохранить**

## Скачать тему

Чтобы скачать тему:

1. На верхней панели нажмите кнопку <span><IconDownload size='xs'/></span>.
1. Выберите формат файла:
   - CSS,
   - JSON,
   - архив CSS+JSON.

Итоговый пресет можно [подключить](##LIBS.LIB.STAND/lib:portal/stand:theme-connecttheme) к любому проекту на Consta.

# Настройка токенов

Токены в Consta создают визуальный язык. Они включают в себя цвета, отступы, шрифт, тени и радиус. В зависимости от ваших настроек токенов автоматически подбирается значение соответствующих переменных. Чтобы посмотреть переменные настройки, в окне настройки нажмите на раскрывающийся список **Переменны**.

Например, настройка `color-base-success` токена **Цвета** будет иметь две переменные: переменная фона `--color-bg-success` и переменная шрифта `--color-typo-succes`.

## Цвета

Набор цветов для создания визуального стиля и согласованного внешнего вида дизайна. Чтобы создать собственную цветовую палитру, меняйте настройки базовых цветов и таким образом задавайте значения цветов в интерфейсе. Подробнее о сущности цветовой палитры читайте
[здесь](##LIBS.LIB.STAND/lib:portal/stand:colorpalette?hash=как-настроить-палитру).

Чтобы настроить цвета:

1. Перейдите на вкладку **Цвета**.
2. Нажмите на цветовую панель в окне настройки и задайте нужный цвет или введите его вручную в поле с кодом цвета.

| Настройка             | Описание                                                                                                    |
| --------------------- | ----------------------------------------------------------------------------------------------------------- |
| $color-base-base      | Базовый цвет содержимого, от которого выстраиваются цвета текста, иконок и элементов управления (контролов) |
| $color-base-essential | Базовый цвет поверхностей                                                                                   |
| $color-base-project   | Проектный цвет, от которого выстраивают акцентные состояния                                                 |
| $color-base-phantom   | Тонирующий цвет, от которого выстраиваются бордеры, оверлей под модальными окнами, скролл                   |
| $color-base-path      | Цвет ссылок и контролов, от которого выстраиваются все их состояния                                         |
| $color-base-system    | Цвет системный                                                                                              |
| $color-base-success   | Успех                                                                                                       |
| $color-base-alert     | Цвет ошибки                                                                                                 |
| $color-base-warning   | Цвет предупреждения                                                                                         |
| $color-base-caution   | Цвет осторожности                                                                                           |
| $color-base-normal    | Цвет нейтральный                                                                                            |
| $color-base-critical  | Цвет критический                                                                                            |

## Отступы

Отступ — это расстояние между элементами интерфейса. При изменении настройки отступа `.Theme_space_gpnDefault` элементы интерфейса адаптируются под заданное расстояние.

## Шрифт

В Consta подключены шрифты [Google Fonts](https://fonts.google.com/specimen/Inter?query=inter). Выберите любой из доступных и настройте его на свой вкус.

| Настройка              | Описание                             |
| ---------------------- | ------------------------------------ |
| Шрифт                  | Доступные шрифты                     |
| .Theme_size_gpnDefault | Размер шрифта и межстрочный интервал |

## Тени

Тень — это градиент или оттенки цвета, который добавляют к элементам интерфейса, чтобы создать эффект глубины и объема. Экспериментируйте с настройками тени, чтобы добиться желаемого эффекта.

| Настройка      | Описание                                                                                                                                                           |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| --shadow-group  | Тень для выделения группы объектов на странице. Используется для карточек.                                                                                         |
| --shadow-layer | Тень для выделения объектов на отдельный слой. Используется для летающих кнопок, боковых панелей, выпадающих списков и временно появляющихся элементов интерфейса. |
| --shadow-modal | Тень для выделения отдельных режимов. Используется в модальных окнах и поповерах.                                                                                  |

Чтобы задать дополнительные настройки (прозрачность, радиус размытия, сдвиг по оси X и Y), нажмите на раскрывающийся список **Дополнительные настройки** и введите нужные значения.

## Радиус

| Настройка        | Описание                             |
| ---------------- | ------------------------------------ |
| --control-radius | Основной радиус скругления контролов |
